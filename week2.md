# Week 2 - Object Detection (YOLO)

## What I did

- Created a virtual environment using Python venv
- Activated the environment
- Installed ultralytics package
- Ran YOLOv8 object detection on a sample image

## Command Used

yolo predict model=yolov8n.pt source="https://ultralytics.com/images/bus.jpg"

## Result

The model detected objects like people and vehicles in the image and generated output with bounding boxes.

## Tools Used

- Python (venv)
- Ultralytics YOLOv8
- macOS Terminal
